= 체크포인트 및 효율성 작업
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
체크포인트는 효율성 작업의 실행 프로세스를 기록하는 데 내부적으로 사용됩니다. 어떤 이유(예: 시스템 중단, 시스템 중단, 재부팅 또는 마지막 효율성 작업이 실패하거나 중지됨)로 인해 효율성 작업이 중지되고 체크포인트 데이터가 있는 경우 최신 체크포인트 파일에서 효율성 작업을 재개할 수 있습니다.

체크포인트가 생성됩니다.

* 작업의 각 단계 또는 하위 단계에서
* 'is stop' 명령을 실행하면
* 기간이 만료되는 경우
