= 액세스 제어 역할 관리 개요
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
관리자에게 할당된 역할에 따라 관리자가 액세스할 수 있는 명령이 결정됩니다. 관리자 계정을 만들 때 역할을 할당합니다. 필요에 따라 다른 역할을 할당하거나 사용자 지정 역할을 정의할 수 있습니다.
