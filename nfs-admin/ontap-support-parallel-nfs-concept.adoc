= ONTAP은 병렬 NFS를 지원합니다
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP는 pNFS(parallel NFS)를 지원합니다. pNFS 프로토콜은 클라이언트가 클러스터의 여러 노드에 분산된 파일 세트에 직접 액세스할 수 있도록 함으로써 성능을 개선합니다. 클라이언트가 볼륨에 대한 최적의 경로를 찾는 데 도움이 됩니다.
